Python virtual environment

  • install 
    pip.exe install virtualenv
  • create 
    python -m virtualenv venv
  • enable 
    .\venv\Scripts\activate
  • install required python-package in virtual environment 
    pip.exe install -r .\requirements.txt
  • deactivate 
    deactivate
  • enable venv in VS Code 
    click on 'Python *version*'-button in the lower left corner and select 'Python *version* - venv' (or similiar)
