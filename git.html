Git Schnippets

  • Undo a commit & redo 
    $ git commit -m "Something terribly misguided" # (0: Your Accident)
$ git reset HEAD~                              # (1)
[ edit files as necessary ]                    # (2)
$ git add .                                    # (3)
$ git commit -c ORIG_HEAD                      # (4)
  • show nice branch graph 
    git log --graph --decorate --oneline
  • show remote-branch-list 
    git branch -vv
  • show history of file 
    git log -p PATH_TO_FILE
  • sync local remote-branch-list with remote 
    git fetch -p
  • list all commits that changed file 
    git log --follow PATH_TO_FILE
  • rename branch 
    git branch -m NEW_BRANCH_NAME
  • add submodule [to specific path] [with tracking specfic branch] 
    git submodule add [-b ]  []
  • update submodule to latest commit of configured branch 
    git submodule update --remote
